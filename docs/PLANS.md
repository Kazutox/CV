# Planning Standards

Status: Verified
Last reviewed: 2026-06-11
Owner: Repository maintainer

## Plan Locations
- Active work plans: `exec-plans/active/`
- Completed work plans: `exec-plans/completed/`
- Debt backlog: `exec-plans/tech-debt-tracker.md`

## Plan Expectations
- Keep plans scoped to concrete changes.
- Include validation commands before implementation starts.
- Move completed plans into `exec-plans/completed/` only after the outcome is recorded.
- For documentation repairs, include the code evidence used to support each major claim.
