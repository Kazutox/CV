## Standards
- Preserve the image-led hero as the first-viewport brand signal.
- Keep repeated content cards compact and scannable.
- Maintain responsive layouts across desktop, tablet, mobile, and print.
- Avoid introducing large unoptimized image assets into the application bundle.
- Use the existing CSS token system before adding new one-off colors or spacing.
