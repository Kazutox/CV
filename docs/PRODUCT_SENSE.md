# Product Sense

Status: Verified
Last reviewed: 2026-06-11
Owner: Repository maintainer

## Product Principle
This repository is a working CV, not a marketing landing page. The first screen should establish identity and credibility quickly, while the rest of the page should make experience, proof points, and contact paths easy to scan.

## Content Principle
Prefer concrete, grounded career evidence from `src/assets/data.json` over generic capability claims.

## Maintenance Principle
The site should remain easy to update by changing structured data and focused components, not by duplicating content across the codebase.
